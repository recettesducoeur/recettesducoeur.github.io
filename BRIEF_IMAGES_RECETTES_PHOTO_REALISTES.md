# Brief maître — Images photo réalistes des recettes

Document de référence pour générer les images des **recettes réelles** du site **Les Recettes du Cœur**.

Ce brief est à utiliser comme base commune pour conserver une cohérence visuelle entre toutes les fiches recettes.

---

## 1. Objectif

Créer une **photo réaliste de recette maison**, simple, appétissante et crédible, pour illustrer une fiche recette du site.

L’image doit donner envie de cuisiner, tout en restant dans un esprit :

- fait maison ;
- simple ;
- familial ;
- accessible ;
- anti-gaspi ;
- chaleureux ;
- non luxueux.

---

## 2. Ligne artistique recommandée

### Usage

Ce style est destiné aux **vraies recettes**.

Pour les éléments génériques du site, conserver le style illustré déjà validé :

- image par défaut de recette ;
- template vierge ;
- page 404 ;
- page “Suggérer une recette” ;
- astuces anti-gaspi ;
- visuels éditoriaux.

### Rendu attendu

- photo réaliste ;
- cuisine maison ;
- lumière naturelle douce ;
- ambiance chaleureuse ;
- présentation simple ;
- plat appétissant mais pas gastronomique ;
- rendu crédible, comme une recette vraiment faite à la maison.

---

## 3. À éviter

Ne pas générer :

- de rendu restaurant haut de gamme ;
- de dressage trop sophistiqué ;
- de vaisselle luxueuse ;
- de décor trop chargé ;
- de style publicité industrielle ;
- de texte dans l’image ;
- de logo ;
- de personnage humain ;
- de mains visibles ;
- de décor sombre ou dramatique ;
- de plat trop parfait ou artificiel.

---

## 4. Format et cadrage

### Format conseillé

- format paysage ;
- image lisible en miniature ;
- plat bien visible ;
- composition claire ;
- fond propre et lumineux.

### Cadrage recommandé

Privilégier :

- vue légèrement de dessus ;
- ou angle 3/4 naturel ;
- plat au centre ;
- quelques ingrédients cohérents autour ;
- ustensiles simples ;
- torchon ou planche en bois si utile.

---

## 5. Ambiance visuelle

L’image doit évoquer une vraie cuisine familiale :

- table en bois clair ;
- assiette simple ;
- bol blanc ou bleu ;
- casserole simple ;
- saladier simple ;
- lumière de fenêtre ;
- ambiance douce ;
- couleurs naturelles.

Palette cohérente avec le site :

- tons clairs ;
- bleu discret ;
- vert naturel ;
- bois clair ;
- touches chaudes alimentaires.

---

## 6. Prompt maître prêt à utiliser

```text
Créer une photo réaliste, chaleureuse et lumineuse d’une recette maison simple pour un site de cuisine solidaire et anti-gaspi.

Le plat doit être présenté de manière appétissante mais accessible, comme une vraie recette faite à la maison. Utiliser une lumière naturelle douce, une table ou un plan de travail clair, une vaisselle simple, et une ambiance familiale propre et conviviale.

Le style doit rester réaliste, non luxueux, non gastronomique, avec un rendu fait maison. Le plat doit être clairement visible, au centre de l’image, avec quelques ingrédients cohérents autour si cela aide à comprendre la recette.

Format paysage, composition propre, lisible en miniature pour une carte recette, sans texte, sans logo, sans personnage humain.
```

---

## 7. Gabarit à adapter pour chaque recette

Remplacer les éléments entre crochets.

```text
Créer une photo réaliste, chaleureuse et lumineuse de [NOM DE LA RECETTE].

Le plat doit ressembler à une recette maison simple, préparée dans une cuisine familiale. Il doit être présenté dans [TYPE DE PLAT OU VAISSELLE], avec une apparence appétissante mais réaliste.

Montrer clairement les ingrédients principaux : [INGRÉDIENTS CLÉS]. Ajouter éventuellement quelques éléments autour du plat : [INGRÉDIENTS OU USTENSILES UTILES].

Ambiance : cuisine maison, lumière naturelle douce, table claire ou plan de travail en bois, vaisselle simple, rendu accessible et anti-gaspi.

Format paysage, cadrage propre, plat bien visible, image lisible en miniature pour une carte recette. Sans texte, sans logo, sans personnage humain, sans style gastronomique trop sophistiqué.
```

---

## 8. Exemple — Salade de riz au thon

```text
Créer une photo réaliste, chaleureuse et lumineuse d’une salade de riz au thon maison.

Le plat doit ressembler à une recette simple, fraîche et familiale, servie dans un grand saladier blanc ou bleu clair. La salade doit contenir du riz, du thon émietté, des tomates, du maïs, du concombre et quelques herbes fraîches.

Ajouter éventuellement autour du plat une boîte de thon ouverte, quelques tomates, un morceau de concombre, une cuillère de service et un torchon simple.

Ambiance : cuisine maison, lumière naturelle douce, table claire ou plan de travail en bois, vaisselle simple, rendu accessible et anti-gaspi.

Format paysage, cadrage propre, plat bien visible, image lisible en miniature pour une carte recette. Sans texte, sans logo, sans personnage humain, sans style gastronomique trop sophistiqué.
```

---

## 9. Constantes à conserver pour toutes les recettes

Toujours conserver :

- photo réaliste ;
- format paysage ;
- lumière naturelle ;
- vaisselle simple ;
- fond clair ;
- ambiance maison ;
- pas de texte ;
- pas de logo ;
- pas de personne ;
- pas de luxe ;
- pas de style gastronomique trop sophistiqué.

Adapter seulement :

- nom de la recette ;
- ingrédients visibles ;
- type de plat ;
- décor secondaire ;
- température visuelle : chaud, froid, gratiné, soupe, dessert ;
- niveau de cuisson visible : doré, crémeux, frais, mijoté, etc.

---

## 10. Conseils recette par recette

### Recette chaude

Insister sur :

- vapeur légère si pertinent ;
- plat juste servi ;
- gratiné ou doré si cuisson au four ;
- lumière chaude mais naturelle.

### Recette froide

Insister sur :

- fraîcheur ;
- couleurs nettes ;
- ingrédients bien identifiables ;
- saladier ou assiette simple.

### Soupe

Insister sur :

- bol simple ;
- texture visible ;
- cuillère ;
- quelques légumes ou herbes autour.

### Dessert

Insister sur :

- rendu maison ;
- portion généreuse mais réaliste ;
- pas de décor pâtisserie professionnelle ;
- vaisselle simple.

### Recette anti-gaspi

Insister sur :

- transformation positive ;
- ingrédients simples ;
- cuisine de placard ;
- reste valorisé sans aspect “restes tristes”.

---

## 11. Checklist avant validation d’une image

Avant d’intégrer une image de recette, vérifier :

- [ ] le plat correspond vraiment à la recette ;
- [ ] les ingrédients principaux sont cohérents ;
- [ ] l’image est appétissante mais réaliste ;
- [ ] le style reste fait maison ;
- [ ] il n’y a pas de texte dans l’image ;
- [ ] il n’y a pas de logo ;
- [ ] il n’y a pas de personnage humain ;
- [ ] l’image reste lisible en miniature ;
- [ ] le format paysage fonctionne pour une carte recette ;
- [ ] l’image ne ressemble pas à une publicité ou à un restaurant gastronomique.

---

## 12. Nommage conseillé des fichiers

Pour rester cohérent avec le site :

```text
images/recettes/[ID_RECETTE].png
images/recettes/[ID_RECETTE].webp
```

Exemple :

```text
images/recettes/20000101-004.png
images/recettes/20000101-004.webp
```

La version `.webp` doit être utilisée dans le site pour l’affichage.

La version `.png` peut être conservée comme source ou archive visuelle.

---

## 13. Résolution et optimisation

Recommandation :

- générer en grand format paysage ;
- conserver une source PNG ;
- convertir en WebP optimisé pour l’affichage ;
- vérifier que la page HTML appelle bien le fichier `.webp`.

Objectif :

- qualité suffisante pour les fiches ;
- poids raisonnable pour un chargement rapide ;
- cohérence avec les optimisations WebP déjà mises en place.
