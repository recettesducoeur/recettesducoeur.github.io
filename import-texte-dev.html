Importer une recette depuis un fichier texte

Cet outil aide à transformer un texte structuré en bloc JSON compatible avec data/recettes.json.

Site statique : cette page ne peut pas écrire directement dans GitHub. Elle génère un JSON à copier ou télécharger, puis à intégrer manuellement dans data/recettes.json.

Format texte attendu

id: 20260708-013
titre: Exemple de recette
categorie: plat
sous categorie: gratin
temperature: chaud
cuisson: four,casserole
preparation: 10
temps cuisson: 20
personnes: 4
ingredients principaux: courgettes|800|g; tomates|500|g
ingredients secondaires: crème|200|ml
ingredients optionnels: fromage|80|g
tags: simple,economique,anti_gaspi
allergenes: lait
resume: Une recette simple.
explication: Explication courte.
etapes: laver les légumes, couper les légumes, cuire au four
astuce: Utiliser les légumes à finir.
variantes: Ajouter du riz ou des pâtes.
visible: true

Pour importer plusieurs recettes dans un même fichier texte, séparez les blocs avec une ligne contenant uniquement ---.
