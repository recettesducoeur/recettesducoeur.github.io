## 🎨 Style

Illustration douce, moderne, lumineuse, aérée, palette officielle, ombres légères, mascottes officielles et aucun texte parasite.

## 📐 Familles

| Type | Référence |
|---|---|
| Hero UI | composition compacte, fond transparent, 1–2 mascottes |
| Illustration éditoriale | 4:3, 3–6 personnages maximum |
| Carte astuce | 16:9, 1–3 personnages |
| Résumé | 16:9, lecture immédiate |
| Placeholder | temporaire et explicitement signalé |

## 🔄 Cycle de validation

```text
Création → contrôle du style et du personnage → nettoyage → validation humaine → export source → WebP optimisé → nommage → intégration → test responsive
```

## 🗃️ Formats

PNG pour source/transparence/documentation ; WebP pour diffusion ; JPEG uniquement si pertinent pour une photographie. Les noms bruts d’outils sont remplacés avant intégration.

## 🧾 Traçabilité minimale

Nom du fichier, scène ou personnage, statut, date d’intégration, référence utilisée et corrections manuelles éventuelles.

## ✅ Statuts

`validated`, `temporary`, `missing`, `archived`.
