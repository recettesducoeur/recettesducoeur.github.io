# Reclassement des ingrédients et recalcul nutritionnel V37

## Changements

- 15 déplacements ou ajouts dans les ingrédients optionnels ;
- les assaisonnements non indispensables ne bloquent plus le calcul ;
- l’eau et le bouillon sont distingués ;
- aucun ingrédient sans quantité n’a été trouvé dans les 13 recettes ;
- les légumes variés restent obligatoires lorsqu’ils définissent réellement la recette.

## Résultat

Avant : **0** recette calculable sur 13.  
Après : **5** recette(s) calculable(s) sur 13.

Les blocages restants proviennent de données nutritionnelles ou de conversions encore absentes dans la base maître des ingrédients.
