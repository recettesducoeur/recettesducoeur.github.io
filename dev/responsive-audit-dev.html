Breakpoints utilisés

  • Desktop : au-dessus de 1100px.
  • Tablette / petit écran : sous 900px.
  • Mobile large : sous 700px.
  • Smartphone : sous 520px.
  • Très petit smartphone : sous 360px.

Les pages dev avec gros tableaux gardent un ascenseur horizontal interne, ce qui est normal : le tableau ne casse pas la page.
