Notes de maintenance

  • Les champs de type liste peuvent évoluer avec les besoins du site.
  • Pour éviter les erreurs de recherche, garder une orthographe stable pour les tags, catégories et types de cuisson.
  • Les recettes test utilisent la date fictive 20000101.

Valeurs de référence ajoutées

Type de cuisson : ajout de sans_cuisson, affiché comme Sans cuisson.

Allergènes : ajout de fruits_a_coque et crustaces, affichés comme Fruits à coque et Crustacés.

Ces valeurs sont aussi documentées dans data/valeurs_reference.json.

Champ PDF ajouté

pdf : chemin relatif vers le PDF statique de la recette.

Format attendu : recettes/pdf/AAAAMMJJ-XXX.pdf

Exemple : recettes/pdf/20260708-011.pdf

Le PDF utilise les quantités de base de la recette. L’impression dynamique PC conserve, elle, les quantités sélectionnées dans le calculateur.
