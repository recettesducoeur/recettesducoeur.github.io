🔎 Recherche

Cherchez par nom de recette, ingrédient, catégorie, cuisson, tags ou filtres rapides. La recherche ignore la casse, les accents et les pluriels simples.

Les boutons de catégorie, cuisson, budget, difficulté, régimes, allergènes, équipement et tags peuvent aussi être cliqués depuis une fiche recette.

Le moteur simplifie les libellés : “thon égoutté” est recherché comme “thon”, “banane mûre” comme “banane”.

Séparez les ingrédients par une virgule, un point-virgule ou une ligne.

Cochez un ou plusieurs tags. Les résultats doivent contenir tous les tags sélectionnés.

⚙️ Filtres rapides

Ces filtres correspondent aux champs cliquables dans les fiches recettes.
