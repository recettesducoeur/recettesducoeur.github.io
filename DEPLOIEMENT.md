# Déploiement GitHub Pages

1. Décompresser le ZIP.
2. Copier tout le contenu du dossier `lesrecettesducoeur_site` à la racine du dépôt.
3. Vérifier que `index.html` et `.nojekyll` sont bien à la racine.
4. Dans GitHub : Settings → Pages → Deploy from a branch → main → /root.
5. Tester l’URL publique.
6. Remplacer `{{URL_FORMULAIRE_SUGGESTION_RECETTE}}` par l’URL réelle du formulaire.
