# Déploiement GitHub Pages — URL racine

Objectif :

https://recettesducoeur.github.io/

Le dépôt GitHub doit s'appeler exactement :

recettesducoeur.github.io

## Point critique

Les fichiers doivent être directement à la racine du dépôt.

On doit voir directement :

- index.html
- 404.html
- assets/
- data/
- dev/
- images/
- qrcodes/
- recettes/
- robots.txt
- .nojekyll

Si le site apparaît sans mise en forme, c'est presque toujours que les dossiers assets/, data/, images/ ou qrcodes/ n'ont pas été envoyés au bon endroit.
