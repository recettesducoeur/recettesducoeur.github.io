# Brief maître — Visuels illustrés du site

Document de référence pour générer les **visuels génériques, éditoriaux et institutionnels** du site **Les Recettes du Cœur**.

Ce brief concerne les images qui ne sont pas des photos réalistes de recettes.

Pour les images de recettes réelles, utiliser plutôt :

```text
dev/BRIEF_IMAGES_RECETTES_PHOTO_REALISTES.md
```

---

## 1. Objectif

Créer des visuels illustrés cohérents pour le site :

- chaleureux ;
- simples ;
- accessibles ;
- solidaires ;
- joyeux sans être enfantins ;
- anti-gaspi ;
- clairs et lisibles en petit format.

Ces visuels servent à renforcer l’identité du site et à rendre les pages plus accueillantes.

---

## 2. Usages concernés

Utiliser ce brief pour :

- image par défaut de recette ;
- template de fiche recette ;
- page 404 ;
- page “Suggérer une recette” ;
- page “Astuces anti-gaspi” ;
- image par défaut des fiches astuces ;
- visuels de présentation du projet ;
- visuels d’aide ou d’explication ;
- illustrations de sections éditoriales ;
- visuels temporaires avant création d’une image dédiée.

Ne pas utiliser ce brief pour :

- les photos réalistes des vraies recettes ;
- les captures d’écran techniques ;
- les logos officiels ;
- les QR codes.

---

## 3. Identité visuelle recherchée

### Style général

- illustration douce ;
- style cartoon propre et moderne ;
- rendu pastel ;
- formes arrondies ;
- ambiance lumineuse ;
- composition claire ;
- personnages ou objets sympathiques, mais pas trop enfantins ;
- aspect “fait maison” et solidaire.

### Sensation à transmettre

L’image doit évoquer :

- cuisine simple ;
- partage ;
- entraide ;
- anti-gaspi positif ;
- cuisine familiale ;
- convivialité ;
- créativité avec peu de moyens ;
- chaleur humaine sans pathos.

---

## 4. Palette recommandée

Couleurs dominantes :

- bleu doux ;
- vert naturel ;
- blanc ou fond très clair ;
- beige / bois clair ;
- touches chaudes : orange doux, jaune pâle, rouge tomate.

À éviter :

- couleurs criardes ;
- fond noir ou très sombre ;
- ambiance dramatique ;
- palette trop luxueuse ;
- rouge agressif dominant ;
- saturation excessive.

---

## 5. Composition recommandée

### Structure

- scène simple ;
- sujet principal bien identifiable ;
- peu d’éléments parasites ;
- image lisible en miniature ;
- fond clair ;
- respiration visuelle suffisante ;
- bon contraste entre sujet et arrière-plan.

### Cadrage

Privilégier :

- format paysage ;
- cadrage frontal ou léger 3/4 ;
- table ou plan de travail clair ;
- objets bien séparés ;
- profondeur simple ;
- contours doux.

### Éléments visuels fréquents

Selon le contexte, utiliser :

- légumes ;
- fruits ;
- casserole ;
- bol ;
- assiette ;
- cuillère en bois ;
- torchon ;
- carnet de recette ;
- boîte à idées ;
- panier ;
- bocal ;
- petit cœur discret ;
- symbole de partage ;
- éléments anti-gaspi : restes valorisés, légumes sauvés, placard, frigo organisé.

---

## 6. Ton graphique

Le ton doit être :

- positif ;
- utile ;
- léger ;
- bienveillant ;
- non culpabilisant ;
- pédagogique ;
- compatible avec un site familial.

Éviter :

- misérabilisme ;
- dramatisation de la précarité ;
- humour trop lourd ;
- style enfantin excessif ;
- style médical ou administratif ;
- esthétique publicitaire trop lisse.

---

## 7. Contraintes importantes

Toujours respecter :

- pas de texte dans l’image, sauf demande explicite ;
- pas de logo intégré dans l’image ;
- pas de marque commerciale ;
- pas de visage réaliste reconnaissable ;
- pas de personnage humain photoréaliste ;
- pas de mains réalistes ;
- pas de nourriture en mauvais état visuellement repoussante ;
- pas de scène de gaspillage sale ou culpabilisante ;
- pas d’élément trop chargé ou difficile à comprendre.

Si un texte est absolument nécessaire, il doit être très court, mais la règle par défaut reste :

```text
Sans texte dans l’image.
```

---

## 8. Différence avec les images de recettes

### Visuels du site

Style recommandé :

- illustration pastel ;
- cartoon doux ;
- identité graphique ;
- usage générique ou éditorial.

### Images de recettes

Style recommandé :

- photo réaliste ;
- plat maison ;
- lumière naturelle ;
- rendu appétissant ;
- pas de style gastronomique luxueux.

Résumé :

```text
Illustrations pour le site.
Photos réalistes pour les recettes.
```

## 9. Prompt maître — Visuel illustré générique

```text
Créer une illustration chaleureuse, lumineuse et conviviale pour le site “Les Recettes du Cœur”, dans un style doux, moderne et légèrement cartoon.

Le visuel doit évoquer la cuisine simple, solidaire et anti-gaspi, avec une ambiance maison, positive et accessible. Utiliser un fond clair, des formes arrondies, une palette cohérente avec le site : bleu doux, vert naturel, blanc, bois clair et quelques touches chaudes alimentaires.

La composition doit être propre, équilibrée et lisible en miniature. Ajouter quelques éléments de cuisine simples et sympathiques : légumes, fruits, bol, casserole, cuillère en bois, torchon, carnet ou panier selon le contexte.

Sans texte, sans logo, sans marque, sans personnage humain réaliste. Style illustré pastel, joyeux, clair et cohérent avec une identité de site de cuisine familiale et anti-gaspi.
```

---

## 10. Gabarit à adapter selon la page

```text

Ajouter quelques éléments cohérents : [OBJETS OU INGRÉDIENTS À AJOUTER].

Le visuel doit rester lisible en miniature, propre, équilibré et chaleureux.

Sans texte, sans logo, sans marque commerciale, sans personnage humain réaliste.
```

## 11. Exemples de prompts par type de visuel

### Image par défaut de recette

```text
Créer une illustration chaleureuse et conviviale pour servir d’image par défaut d’une fiche recette lorsque la photo réelle est absente.

Représenter une scène de cuisine maison lumineuse avec un plat générique fait maison au centre, sans faire référence à une recette précise. Ajouter quelques éléments simples autour du plat : cuillère en bois, bol, petite casserole, légumes simples, herbes et torchon de cuisine.

Style illustré doux, moderne, légèrement cartoon, fond clair, palette bleu doux et vert naturel avec touches chaudes. Image lisible en miniature, équilibrée et accueillante.

Sans texte, sans logo, sans personnage humain réaliste.
```

### Page 404

```text
Créer une illustration humoristique et douce pour une page 404 d’un site de cuisine solidaire.

Représenter une recette qui s’est échappée ou une assiette vide avec quelques miettes, dans une ambiance légère et amusante. Le visuel doit rester chaleureux, non dramatique, clair et compatible avec l’identité du site.

Style illustré pastel, fond clair, formes arrondies, palette bleu doux et vert naturel avec touches chaudes.

Sans texte, sans logo, sans marque, sans personnage humain réaliste.
```

### Page “Suggérer une recette”

```text
Créer une illustration conviviale pour une page “Suggérer une recette”.

Représenter une boîte à idées ou un carnet de recettes ouvert, entouré de quelques ingrédients simples, d’un crayon, d’un bol et d’un torchon. L’image doit évoquer la participation, le partage, la cuisine maison et les idées anti-gaspi.

Style doux, pastel, moderne, légèrement cartoon, fond clair, palette bleu doux et vert naturel.

Sans texte, sans logo, sans personnage humain réaliste.
```

### Astuces anti-gaspi

```text
Créer une illustration joyeuse et pédagogique pour une page d’astuces anti-gaspi.

Représenter des aliments simples sauvés du gaspillage : légumes, fruits, pain, bocaux, restes bien rangés, avec quelques ustensiles de cuisine. L’ambiance doit être positive, propre, utile et non culpabilisante.

Style illustré pastel, fond clair, formes arrondies, palette bleu doux, vert naturel et touches chaudes.

Sans texte, sans logo, sans personnage humain réaliste.
```

### Aide / fonctionnement du site

```text
Créer une illustration claire et pédagogique pour une page d’aide d’un site de recettes.

Représenter une cuisine bien organisée avec un carnet, quelques fiches recettes, une loupe, un QR code stylisé non lisible, des ustensiles et des ingrédients simples. Le visuel doit évoquer la recherche, l’organisation et l’utilisation facile du site.

Style doux, lumineux, légèrement cartoon, fond clair, couleurs bleu doux et vert naturel.

Sans texte lisible, sans logo, sans marque, sans personnage humain réaliste.
```

## 12. Visuels temporaires et placeholders

Pour un placeholder, privilégier :

- scène générique ;
- aucun ingrédient trop spécifique ;
- pas de recette identifiable ;
- message visuel simple ;
- bon rendu en carte miniature.

Un placeholder doit pouvoir fonctionner dans plusieurs contextes sans créer d’incohérence.

Exemples :

- plat générique maison ;
- panier d’ingrédients ;
- table de cuisine simple ;
- carnet de recettes ;
- bol et légumes ;
- restes transformés positivement.

## 14. Export et optimisation

Recommandation :

- conserver un PNG source ;
- créer une version WebP optimisée ;
- utiliser le WebP dans les pages HTML ;
- garder le PNG comme archive ou source de secours ;
- vérifier le poids des fichiers ;
- éviter les images trop grandes si elles sont seulement utilisées en carte.

Objectif :

- affichage rapide ;
- cohérence visuelle ;
- bonne qualité sur desktop et mobile ;
- compatibilité avec l’optimisation actuelle du site.

---

## 15. Checklist de validation

Avant d’intégrer un visuel illustré, vérifier :

- [ ] le style correspond aux visuels déjà validés ;
- [ ] l’image est claire en miniature ;
- [ ] les couleurs restent cohérentes avec le site ;
- [ ] le fond est clair ;
- [ ] le visuel n’est pas trop chargé ;
- [ ] il n’y a pas de texte non demandé ;
- [ ] il n’y a pas de logo intégré ;
- [ ] il n’y a pas de marque commerciale ;
- [ ] le ton reste positif et non culpabilisant ;
- [ ] l’image fonctionne en carte et en page ;
- [ ] la version WebP existe ;
- [ ] la page HTML appelle bien le fichier WebP.

---

```text
Les visuels génériques racontent l’esprit du site.
Les photos de recettes montrent le plat.
```
